# Chifrijo

## Ingredientes

1. Arroz
2. Cubaces
3. Chicharon
4. Chimichurri
5. Aguacate
6. Piquitos
7. Chile Jalapeño
8. Birra

## Procedimiento

- Se pica en trozos el chicharron.
- Se mezclan el arroz y los **_cubaces_**.
- Se agrega el _aguacate_ y **chimichurri**.
- Se agrega el _chile_.
- Se colocan los **_piquitos_**
- Se destapa la **birra**
